#+TITLE: Architecture Workspace

* Purpose

This workspace is dedicated exclusively to architectural reasoning and
design convergence.

It is not an implementation tracker. Do not use it for bugs, coding tasks,
TODO lists, CI failures, refactoring chores, or implementation progress.

Workspace discussions preserve how architectural decisions were reached.
Stable decisions are migrated into authoritative ADRs, design documents,
or specifications.

* Lifecycle

OPEN -> DISCUSSING -> DECIDED -> MIGRATED

- OPEN: An architectural question has been identified.
- DISCUSSING: Alternatives and trade-offs are under active discussion.
- DECIDED: The architectural direction has converged.
- MIGRATED: The stable decision has been incorporated into an authoritative
  ADR, design document, or specification.

Workspace files are retained after migration as design history.
